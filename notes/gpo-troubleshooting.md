# GPO Troubleshooting

## Common commands

```cmd
gpupdate /force
gpresult /r
```

## Common issues

- Incorrect OU placement
- Replication delays
- DNS problems
- Security filtering
- Client not receiving policies

## Troubleshooting steps

1. Verify domain connectivity
2. Check DNS configuration
3. Run gpupdate /force
4. Review applied policies
5. Check event logs
